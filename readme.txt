== Description ==

### Ready to Be the First Page of Google?
AI-based WordPress tag generation and internal links plugin.

__See SEO Gun in action:__

[youtube https://www.youtube.com/watch?v=eGvTRJ67czc]

__SEO Gun Main Features:__

* __" Automation "__ Don't waste time looking for post tags or find internal links we made it for you.

* __" AI-Based Plugin "__ Our well trained AI servers will find the best keywords and entities for your article.

* __" Google Trends "__ Our plugin will find if any of your article words is related to Google Trends for the last 12 months and will add it as a post tag and internal link.

* __" WordPress Tags "__ Our plugin will match your post with all your previous WordPress tags.

* __" Entities "__  Our AI system will scan your post content and will extract the entitles like "Events name and Places and Person names" and will make tags and internal links based on it.

* __"Link Value"__ Building an SEO-friendly site architecture with internal links, This will help google to give your article more value.

* __"Decrease bounce rate"__ Say goodbye to high bounce rates, Our plugin will let the user browse more pages and stay more on your website.

* __"Increase your advertising revenue"__ Internal links will let the user browse more pages which means more ad impressions and more money.

== Installation ==

How to install the plugin

1. Upload the plugin files to the `/wp-content/plugins/seogun` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' screen in WordPress

3. Use the Settings->SEO GUN screen to configure the plugin.


== Frequently Asked Questions ==

= How it works? =

Simply, Once you finish writing your new article/post click on generate keywords button and our AI system will try to find the most related keywords to your article and add it as post tag and internal links.

= Why I should care about internal links? =

Internal links have an extremely high value when it comes to SEO and also the general optimization of websites.

= Will SEO Gun slow down my website? =

The answer is No, Our plugin works on admin panel only and it does not affect your website speed.

= Can I use SEO Gun with Yoast? =

Yes, Our plugin is compatible with Yoast SEO.

= Will this plugin overwrite links which I already set manually? =

No. If you have already created a link manually, the plugin will not overwrite it.
